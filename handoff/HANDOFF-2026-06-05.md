## The 4 Broken Patches

### 1. `no-layout-position` — OBSOLETE (delete)

**Bundle:** thread (`local-conversation-thread-D4J7KD_b.js`)
**What happened:** Codex removed `layout:"position"` entirely. Current layout values are only `"horizontal"`, `"verticalIcon"`, `"vertical"`.
**Fix:** Delete the patch entry from PATCHES in `source-patcher.js`. Remove it from `SETTING_FEATURES["show-reasoning"]`. The patch is dead code.

### 2. `reasoning-no-animate-height` — SKELETON BUG (one-char fix)

**Bundle:** thread
**Current skeleton:** `match: /initial:!1,animate:P,transition:(\w+)/`
**Problem:** Hard-codes `animate:P` (literal P) instead of `animate:\w`. Variable renamed from `P` to `R` in v26.602.
**Actual code:** `initial:!1,animate:R,transition:Vo` (3 matches in bundle)
**Fix:** Change skeleton match from `animate:P` to `animate:\w`:
```js
// source-patcher.js, reasoning-no-animate-height skeleton
match: /initial:!1,animate:\w,transition:(\w+)/,
```
Also update the `replacement` function to use `animate:\w` and the `unpatched` regex similarly.

### 3. `reasoning-no-autocollapse` — PATTERN REMOVED (needs reverse-engineering)

**Bundle:** thread
**Original pattern:** `if(!o){S(!1);return}` — where S=setExpanded(false), called when reasoning finishes.
**What happened:** Zero matches for this pattern in the current bundle. Codex rewrote the accordion collapse logic entirely.
**Fix needed:** Reverse-engineer the new collapse-on-finish code in `local-conversation-thread-D4J7KD_b.js`. Look for:

### 4. `thought-fade-disable-un` — FUNCTION RENAMED (needs new skeleton)

**Bundle:** markdown
**Original pattern:** `function Un({content:e,cwd:t,fadeText:n,...}){...key:n?void 0:...}`
**What happened:** The `Un` function is gone from all markdown bundles. Found `function ir({fadeText:e` in `markdown-CLVNcNL7.js` — function renamed from `Un` to `ir`, parameter list likely changed.
**Fix needed:** Extract the new function signature from `markdown-CLVNcNL7.js` and write a new skeleton. The key behavior to patch: remove the `fadeText` conditional that wraps content in animated `<span>` elements.
